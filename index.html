### Description This pen shows how to leverage the CosmicLink format to pass a transfer request from a simple HTML form. ### Keypoints - Because it doesn't need any library, this pen is the lightest transfer form ever made! - The HTML form builds a CosmicLink query that is passed to the Cosmic.link website. - CosmicLink queries syntax has been derived from `js-stellar-sdk` operations syntax. - It is possible to create multi-operation transactions using the syntax `?type=transaction&{...metaFields|txFields}&operation={type1}&{...op1Parameters}...&operation={typeN}&{...opNParameters}` - By leveraging the multi-operation syntax and hidden inputs, it is possible to create advanced transaction templates. ### Challenges - **Easy/** Transform the HTML form into a donation form, that let the user send any amount to an address of your choice. - **Hard/** Edit the form into one that let the user open a trustline for a selection of assets. _Give it a try by editing the code, and be the first to post a solution on GalacticTalk!_ ### Documentation - CosmicQueries cheatsheet - StellarSdk Operations specification ### Social - [Discuss that pen on GalacticTalk](https://galactictalk.org/d/2122-pen-of-the-week-html-only-transfer-request-form)
